# Snowflake POC

## Steps

1. Clone the repo
2. Run npm install
3. Create a .env file in root and add following:

accessUrl="<your-access-url>"
authenticator="<your-authenticator>"
account="<your-account>"
username="<your-username>"
password="<your-password>"
database="<your-database>"
schema="<your-schema>" 
  
4. Run node index.js
